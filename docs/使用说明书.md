# Task Nexus 使用说明书

## 软件概述
Task Nexus 是一款 Windows 游戏优化工具，通过 CPU 核心调度、内存管理和系统调优来提升游戏性能。

---

## 文件位置

| 文件类型 | Windows 路径 |
|----------|--------------|
| **配置文件** | `%APPDATA%\task-nexus\config.json` |
| **日志文件** | `%APPDATA%\task-nexus\logs\app.log` |

> 💡 快速访问：Win+R → 输入 `%APPDATA%\task-nexus` → 回车

---

## 功能说明

### 一、核心调度页面

| 功能 | 说明 |
|------|------|
| **进程扫描** | 扫描当前运行的进程，按 CPU 占用率排序 |
| **核心选择** | 勾选要分配给游戏的 CPU 核心 |
| **热力图** | 实时显示每个核心的 CPU 占用率 |
| **优先级设置** | 设置进程优先级（实时/高/正常/低等） |
| **批量操作** | 多选进程后一键设置优先级 |

---

### 二、游戏模式页面

| 功能 | 说明 |
|------|------|
| **T mode1** | 默认模式，系统自动调度 |
| **T mode2** | 笔记本优化模式，降低温控掉帧 |
| **T mode3** | 极致性能模式，最高优先级 + 核心隔离 |
| **自动化策略** | 保存配置，下次自动应用 |
| **后台压制** | 游戏运行时自动降低后台进程优先级 |
| **ProBalance** | 自动压制高 CPU 后台进程 |

---

### 三、一键优化页面

| 功能 | 说明 |
|------|------|
| **系统优化** | 一键应用网络/内存/性能优化 |
| **内存清理** | 清理系统缓存释放内存 |
| **电源计划** | 快速切换高性能电源计划 |
| **高精度计时器** | 降低输入延迟（仅对调度进程生效） |
| **开机自启** | 系统启动时自动运行 |
| **托盘驻留** | 关闭窗口时最小化到托盘 |
| **配置导入/导出** | 分享或备份配置文件 |

---

## 激活系统

### 用户激活流程
1. 首次启动显示激活对话框
2. 复制**机器码**发给卖家
3. 卖家返回**激活码**
4. 输入激活码点击激活

### 卖家生成激活码
1. 打开 `tools/激活码生成器.html`
2. 粘贴用户机器码
3. 点击生成，复制激活码发给用户

---

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 看不到激活对话框 | 删除 `%APPDATA%\task-nexus\config.json` 后重启 |
| 软件卡顿 | 检查日志文件 `logs\app.log` 定位问题 |
| 优化不生效 | 以管理员身份运行软件 |
| 换电脑后无法使用 | 需要重新获取激活码 |
